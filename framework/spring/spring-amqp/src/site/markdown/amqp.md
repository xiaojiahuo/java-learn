# AMQP 0-9-1 学习

## 1 简介
> 高级消息队列协议（AMQP）是面向消息中间件的开放标准应用层协议。 AMQP的定义特征是消息定向，排队，路由（包括点对点和发布和订阅），可靠性和安全性。
